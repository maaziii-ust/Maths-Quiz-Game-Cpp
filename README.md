# Maths Quiz Game (C++)

## Overview

Maths Quiz Game is a console-based educational game developed in C++. The game challenges players with randomly generated mathematical questions across multiple difficulty levels. Players earn points for correct answers and progress through increasingly difficult levels.

## Features

* 5 Difficulty Levels
* Random Question Generation
* Addition, Subtraction, Multiplication and Division
* Powers and Square Roots
* Scoring System
* Life System
* Level Progression
* Interactive Console Interface

## Technologies Used

* C++
* Structures
* Functions
* Random Number Generation
* Loops and Conditional Statements

## Game Rules

* Start with 4 lives.
* Each level contains 5 questions.
* Answer at least 3 questions correctly to pass the level.
* Wrong answers reduce lives.
* Reach Level 5 to complete the game.
